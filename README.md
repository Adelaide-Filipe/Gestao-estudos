# 📚 API de Gestão de Tarefas de Estudo

Este projeto é uma API REST desenvolvida em Node.js, totalmente containerizada, para a gestão de rotinas de estudo. Faz parte da avaliação do **Checkpoint Fase 1 da Formação em que estou**.

## 🚀 Funcionalidades (Checkpoint 1)
O projeto cumpre os seguintes requisitos técnicos:
1. **Configuração Dinâmica:** Uso de variáveis de ambiente (`.env`).
2. **Endpoints HTTP:** Implementação de rotas para Listagem e Criação de tarefas.
3. **Logs Estruturados:** Monitorização via console em formato JSON (`JSON.stringify`).
4. **Orquestração:** Configuração completa via Docker Compose.
5. **Persistência:** Uso de volumes Docker para persistir dados num ficheiro JSON local.
6. **Versionamento:** Repositório Git com commits seguindo o padrão Conventional Commits e histórico estruturado de desenvolvimento.

## 🛠️ Tecnologias
- **Node.js** & **Express**
- **Docker** & **Docker Compose**
- **Git** (Estratégia de Branching)
